# Tiva_Mikroislemci_Odevleri

# Tiva TM4C123GH6PM Mikrodenetleyici Ödevleri

**Ad Soyad:**Ahmet Taha Karataş	
**Öğrenci No:**G210100080
**Ders:**İleri Mikroişlemciler

Bu depo, dönem içi verilen 4 adet mikrodenetleyici ödevini içermektedir.

## Ödev Listesi

### [Ödev 1: 16x2 LCD Sürücü Yazımı](./Odev1_LCD_Driver)
Tiva C GPIO pinleri kullanılarak 16x2 LCD ekranın sürülmesi.
- **Durum:** Tamamlandı

### [Ödev 2: LCD Saat Uygulaması](./Odev2_Digital_Clock)
Timer kesmeleri kullanılarak oluşturulan dijital saat uygulaması.
- **Özellik:** Systick/Timer0 Interrupt ile 1 saniye hassasiyet.

### [Ödev 3: ADC Okuma](./Odev3_LCD_ADC)
Potansiyometre verisinin ADC modülü ile okunup LCD'ye yazdırılması.

### [Ödev 4: UART ve GUI Kontrolü](./Odev4_Serial_GUI)
Bilgisayar arayüzü ile kart arasında seri haberleşme.
